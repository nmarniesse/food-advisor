# Food advisor API

API to generate menus using ChatGPT

## Requirements

- Go
- SQLite3

## Usage

- Get an API key on https://platform.openai.com/api-keys
- Copy .env.dist to .env
- Fill the CHATGPT_TOKEN in your .env
- Run the server and go to http://localhost:8080

## Run API in local

```bash
go run .
```
